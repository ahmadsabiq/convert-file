# convert-file

Convert File Apps, PDF, Excel, Docs, PPTX, etc.

## Features

- **PDF to Excel Converter**: Extract and edit tables from PDF files.
- **CSV Converter**: Clean, view, edit and convert CSV data.
- **File Compressor**: Compress various document files.
- **File Merger**: Merge multiple documents into a single file.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

Install dependencies:

```bash
npm install
```

### Running Locally

To run the development server:

```bash
npm run dev
```

### Build for Production

To build the application for production:

```bash
npm run build
```
